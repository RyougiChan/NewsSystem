## 简介
- Java 实现的文章发布管理系统，作为未来个人网站文章管理模块提前进行了开发
- 进行了前后端和数据库整体开发

## 技术支持
* **Spring4** + **Hibernate4** + **Struts2**
* **JAVA** (With jdk version  "1.8.0_91" Java(TM) SE Runtime Environment (build 1.8.0_91-b15))
* **JavaScript/JQuery**
* **HTML/JSP + CSS**
